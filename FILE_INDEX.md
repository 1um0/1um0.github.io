## 📞 快速问题排查

| 问题 | 可能原因 | 检查文件 |
|------|---------|---------|
| 页面不显示 | Web 服务器未启动 | QUICKSTART.md |
| 登录失败 | 账号/密码错误 | README.md 中的默认账号 |
| 样式错乱 | CSS 未加载 | 检查 `<link>` 标签 |
| 成员页面损坏 | 路径不正确 | `pages/member-*.html` |
| 数据丢失 | localStorage 被清除 | SETUP.md 调试部分 |
| 上传文件失败 | 未配置云存储 | `js/storage-config.js` |
